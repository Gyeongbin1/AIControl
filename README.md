## 나
저의 이름은 **박 경 빈** 입니다.
저는 마산합포구에 연세병원 근처에 살고 있습니다.

## 가 족 구 성 원
- 저의 가족은 아버지, 어머니, 형, 나, 강아지 이렇게 4명과 강아지 1마리로 구성 되어있습니다.
- 아버지는 건설쪽에서 일을 하십니다.
- 어머니는 주부입니다.
- 형은 현대로템에서 일을 하고 있습니다.
- 강아지 종은 믹스견입니다.

## 학 력
- 초등학교 : 월영초등학교 졸업
- 중학교   : 마산중학교 졸업
- 고등학교 : 마산공업고등학교 메카트로닉스과 졸업
- 대학교   : 한국폴리텍대학 창원캠퍼스 스마트팩토리 재학
